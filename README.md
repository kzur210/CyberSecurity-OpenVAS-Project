# CyberSecurity-OpenVAS-Project
Semester project: Vulnerability scanning using OpenVAS
This project demonstrates a controlled vulnerability assessment using OpenVAS (Greenbone Vulnerability Manager).

Project Highlights:
Configured an isolated lab environment with Metasploitable2, an intentionally vulnerable virtual machine.
Performed network scanning, service enumeration, and vulnerability detection using OpenVAS on Linux.
Generated comprehensive reports analyzing potential risks and configuration issues.
Documented challenges and implemented solutions to improve scanning methodology and analysis accuracy.

Project Report: https://github.com/kzur210/CyberSecurity-OpenVAS-Project/blob/main/CyberSecurity-ProjectReport.pdf

All work performed in a controlled lab environment for educational purposes only. No real-world systems were affected.
